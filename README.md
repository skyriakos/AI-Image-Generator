# AI-Image-Generator
A modern desktop application built with Python and Tkinter that allows you to generate images using OpenAI's `gpt-image-1` model. Features a sleek GUI with a Home Screen, input prompt field, status messages, and image preview.

## Features

- **Home Screen:** Welcome page with description and start button.
- **Prompt Input:** Enter a prompt for the AI to generate an image.
- **Image Generation:** Generates images via OpenAI API and displays them directly in the app.

## Requirements

- Python 3.8+
- OpenAI API Key(https://platform.openai.com/account/api-keys)
